About programming

Programming is all about writing code. Code can be simply defined as a set of instructions that a computer can understand.

But computers cannot understand code in itself, because it speaks only binary! So, we need a translator; in this case, the translator is called a compiler.

Some programming languages are not compiled to binary code. Instead, they are interpreted, This means that the translator, i.e., interpreter, translates the code to machine instructions in real time. This can be a bit slow, but it has some advantages.
